# Secure File Monitoring System

## 📌 Description
This project monitors file integrity using SHA-256 hashing and detects unauthorized changes.

## 🎯 Objectives
- Monitor file integrity
- Detect file modifications
- Generate alerts

## 🛠️ Tools Used
- Python
- hashlib

## ▶️ How to Run
python3 file_monitor.py

## 📊 Output
- Alerts when file changes
- Displays file hash
